# Skill: travel-planner

旅行・おでかけの計画を、**行程表 / 持ち物リスト / 予算**の 3 点セットで作る。

## Step 1. ヒアリング（1 回で聞き切る）

不足している項目だけを **1 メッセージにまとめて**聞く（質問攻めにしない）。
分かっている項目は聞き直さない。

- 行き先・時期（未定なら候補出しから手伝う）
- 日数・人数・同行者（子供・高齢者・ペットの有無は行程の密度に直結）
- 移動手段（車 / 電車 / 飛行機）と出発地
- 予算感（ざっくりで可）
- 目的・優先度（観光 / グルメ / 休養 / イベント）と、外せない予定
- 体力・ペースの希望（詰め込み型 / ゆったり型）

回答が部分的でも「仮置き」を明示してプランを出す（完璧な情報を待たない）。

## Step 2. 行程表を作る

```markdown
## 📅 行程表: <行き先>（N泊M日）

### Day 1（M/D 曜）

| 時間  | 予定           | メモ（移動時間・予約要否・雨天代替） |
| ----- | -------------- | ------------------------------------ |
| 09:00 | 出発（〜駅）   | 特急◯◯、事前予約推奨                 |
| 12:00 | 昼食: ◯◯エリア | 候補2〜3軒                           |
```

### 行程の質の基準

- **移動時間を現実的に見積もる**（乗換・チェックイン・混雑を含める）。
  1 日に主要スポットは 2〜4 個まで。詰め込みすぎは指摘して間引く
- 食事はエリアで押さえ、候補を 2〜3 出す（1 軒に賭けない）
- **雨天・休業の代替案**を各日に 1 つ添える
- 定休日・営業時間・季節要因（紅葉時期・海開き等)は「要確認」と明示する。
  **未確認情報を確定事実として書かない**（Web 検索が使える環境なら確認してから書く）
- 子連れ・高齢者連れなら昼寝・休憩・トイレの余白を組み込む

## Step 3. 持ち物リストを作る

カテゴリ別チェックリスト形式。行き先・季節・移動手段・同行者に合わせて増減する。

```markdown
## 🎒 持ち物リスト

### 必須（忘れたら詰む）

- [ ] 財布・交通系IC / 免許証
- [ ] スマホ + 充電器 / モバイルバッテリー
- [ ] 予約確認（チケット・宿）
- [ ] 健康保険証（コピー可）・常備薬

### 行き先・季節対応

（海外なら: パスポート・ビザ・変換プラグ・海外対応SIM/Wi-Fi）
（冬山・海・温泉など、文脈に合わせて）

### あると快適

（圧縮袋・エコバッグ・ジップロック・折りたたみ傘 など)
```

- 「必須（忘れたら詰む)」を最上位に分離する
- 現地調達できるものは「現地で買える」と添えて荷物を減らす方向に倒す

## Step 4. 予算概算を作る

```markdown
## 💰 予算概算（N人 / N泊M日）

| 項目       | 概算           | 前提                   |
| ---------- | -------------- | ---------------------- |
| 交通       | ¥--,--         | 往復◯◯ + 現地移動      |
| 宿泊       | ¥--,--         | 1泊¥--,-- × N泊 × N室  |
| 食事       | ¥--,--         | 朝昼夜の相場から       |
| 観光・体験 | ¥--,--         | 入場料・アクティビティ |
| 予備       | 全体の 10〜15% | 想定外・お土産         |
| **合計**   | **¥--,--**     |                        |
```

- 相場は時期・地域で大きく振れるため「概算」であることと前提を明示する
- ユーザーの予算感を超えたら、削れる項目の優先順位（宿のグレード / 移動手段 / 日数）を提案する

## Step 5. 仕上げ

- 予約が必要なもの（宿・特急・レストラン・体験）を **期限付き TODO** として最後にまとめる
- 「出発 1 週間前チェック」（天気予報・持ち物最終確認・充電）を添える
- 修正依頼には該当セクションだけ差し替えて全体を再掲する（部分レスポンスで散らかさない）

## やってはいけないこと

- 営業時間・料金・運行情報を未確認のまま断定する（「要確認」を付ける）
- 移動時間を楽観で積んだ非現実的な行程
- ヒアリングを何往復にも分ける（1 回でまとめて聞く）
- 予算の合計だけ出して内訳・前提を省く
