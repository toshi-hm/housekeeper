# Skill: cleaning-planner

掃除・片付けを「続けられる計画」に落とす。単発の大掃除と、日常ルーティンの両方に使う。

## Step 1. ヒアリング（1 回で聞き切る）

不足分だけを 1 メッセージで聞く:

- 住まいの規模（間取り）と気になる場所
- 目的: 日常ルーティン化 / 大掃除（年末・引っ越し・来客前）/ 特定箇所の攻略
- 使える時間（平日◯分・週末◯時間）と担当者（1人 / 家族分担）
- 苦手・後回しにしがちな場所
- ペット・小さい子供の有無（洗剤選び・優先度に影響）

## Step 2-A. 日常ルーティンの場合

**頻度 × 場所のマトリクス**でスケジュール化する。

```markdown
## 🧹 掃除ルーティン

### 毎日（合計10分以内に収める）

- [ ] 食後のシンクリセット（2分）
- [ ] 床のものを定位置に戻す（5分）

### 週次（曜日に固定して習慣化）

| 曜日 | 場所 | 時間 | 内容         |
| ---- | ---- | ---- | ------------ |
| 月   | 風呂 | 15分 | 浴槽・排水口 |

### 月次

（換気扇フィルター・冷蔵庫内・窓 など）

### 季節・年次

（エアコン・カーテン洗濯・ベランダ など）
```

### 設計の原則

- **毎日の負荷は合計 10 分以内**。続かない計画は失敗した計画
- 週次は曜日固定（「いつかやる」を消す)。家族分担なら担当列を足す
- 「汚れる速度」で頻度を決める（水回り高頻度、寝室低頻度）
- 完璧主義を仕込まない。「8割きれいを維持」を目標に置き、リカバリ日（サボった週の敗者復活）を用意する

## Step 2-B. 大掃除・単発の場合

**段取り表 + 場所別チェックリスト**を作る。

```markdown
## 🧽 大掃除プラン（総見積: N時間 / N日）

### 事前準備（買い出しリスト）

- [ ] 洗剤類: （場所に応じて。混ぜるな危険の注意も）
- [ ] 道具: ゴミ袋・古歯ブラシ・マイクロファイバー等

### 段取り（順番に意味がある）

1. 不用品処分・片付け（掃除の前に物を減らす）
2. 上から下へ（照明・棚 → 床）、奥から手前へ
3. 洗剤の漬け置きが要る場所（換気扇・排水口）を最初に仕掛けて、待ち時間に他をやる

### 場所別チェックリスト

#### キッチン（見積: N分）

- [ ] 換気扇: （手順 2〜3 行 + 使う洗剤）
```

### 設計の原則

- 漬け置き・乾燥の**待ち時間を並列化**した順序にする（正味時間を圧縮）
- 1 日に詰め込みすぎない。体力配分（重労働は午前）まで考える
- 見積時間を場所ごとに書く（全体像が見えるとサボりにくい）

## Step 3. 手順・洗剤の知識を添える

- 場所ごとに「使う洗剤 / やり方の要点 / やってはいけないこと」を 2〜3 行で
- **安全注意は必ず書く**: 塩素系×酸性系は混ぜない（致死性ガス）、換気、ゴム手袋、
  素材を傷める組み合わせ（アルミにアルカリ、大理石に酸 等）
- 「プロに頼んだ方が安い/安全」なライン（エアコン内部・高所外窓等）は正直にそう言う

## Step 4. フォローアップ

- ルーティンの場合: 「2 週間回してみて、できなかった項目を教えてください。
  計画の方を現実に合わせて削ります」と締める（人ではなく計画を直す）
- 定期リマインドを望まれたら、Claude Code の定期実行（Routine / スケジュール機能）を提案する

## やってはいけないこと

- 実行不可能な理想プラン（毎日 1 時間掃除、など生活に合わない計画）
- 洗剤の安全注意の省略（特に塩素系・酸性系の併用禁止）
- 素材を確認せずに強い洗剤・研磨を勧める
- サボりを責めるトーン（計画側を軽くするのがこのスキルの仕事）
