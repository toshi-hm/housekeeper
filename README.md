# housekeeper

自宅の備品・食料品を管理するウェブアプリ — A home inventory management app for tracking household goods and groceries.

## Tech Stack

- **Build**: Vite 8
- **Language**: TypeScript (strict mode)
- **UI**: React 19
- **Routing**: TanStack Router (file-based)
- **DB/Auth**: Supabase (Postgres + RLS + Supabase Auth)
- **UI Kit**: Tailwind CSS v4 + shadcn/ui
- **Lint**: oxlint
- **Validation**: Zod
- **Server State**: TanStack Query v5
- **Hosting**: Cloudflare Pages

## Features

- 📷 **Barcode scanning** — scan product barcodes via device camera
- 📦 **Product registration** — name, barcode, category, image
- 🗂 **Inventory management** — quantity, location, purchase/expiry dates, notes
- ⚠️ **Expiry alerts** — visual highlights for items expiring within 3 days or already expired
- 🔐 **Authentication** — Supabase Auth (email/password)

## Getting Started

### 1. Set up Supabase

1. Create a new project at [supabase.com](https://supabase.com)
2. Run the migration in the SQL Editor:
   ```
   supabase/migrations/20240101000000_create_items.sql
   ```

### 2. Deploy Supabase Edge Function

The barcode lookup is proxied through a Supabase Edge Function to avoid CORS issues.

```bash
# Install Supabase CLI if not already installed
# https://supabase.com/docs/guides/cli

supabase login
supabase link --project-ref your-project-ref
supabase functions deploy barcode-lookup
```

### 3. Configure Environment Variables

```bash
cp .env.example .env.local
```

Edit `.env.local` with your Supabase credentials:

```
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

### 4. Install and Run

```bash
bun install
bun run dev
```

### 5. Build

```bash
bun run build
```

### 6. Deploy to Cloudflare Pages

Connect your GitHub repository to Cloudflare Pages with:

- **Build command**: `bun run build`
- **Build output directory**: `dist`
- **Environment variables**: `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`

## Development

```bash
bun run dev        # Start dev server
bun run build      # Build for production
bun run typecheck  # Type check only
bun run lint       # Lint with oxlint
bun run preview    # Preview production build
```

## Project Structure

```
src/
  routes/
    __root.tsx                      # Root layout
    _auth.tsx                       # Auth layout (redirects to /login if not authenticated)
    _auth.index.tsx                 # Dashboard / item list
    _auth.items.new.tsx             # Add new item
    _auth.items.$itemId.tsx         # Item detail
    _auth.items.$itemId.edit.tsx    # Edit item
    login.tsx                       # Login / signup
  components/
    ui/                             # shadcn/ui components
    BarcodeScanner.tsx              # Camera-based barcode scanner
    ItemCard.tsx                    # Item summary card
    ItemForm.tsx                    # Create/edit item form
    ExpiryBadge.tsx                 # Expiry status badge
  lib/
    supabase.ts                     # Supabase client
    queryClient.ts                  # TanStack Query client
    utils.ts                        # Utility functions
  hooks/
    useItems.ts                     # Item CRUD hooks
    useBarcodeLookup.ts             # Product lookup from barcode
  types/
    item.ts                         # Item types and Zod schemas
  routeTree.gen.ts                  # Auto-generated by TanStack Router
supabase/
  functions/
    barcode-lookup/
      index.ts                       # Edge Function: proxy barcode lookup to Open Food Facts
  migrations/
    20240101000000_create_items.sql  # Database schema
```
