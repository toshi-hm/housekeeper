プライバシーポリシー / Privacy Policy

最終更新 / Last updated: 2026-05-24

1. スキルの概要

「ハウスキーパー」は、個人が自宅の在庫管理を行うための Alexa スキルです。 本スキルは個人利用のみを目的としており、Alexa スキルストアへの公開を行っていません。

2. 収集する情報

本スキルが収集・保存する情報は以下のみです：

  • ユーザー自身が入力した在庫データ（商品名・数量・賞味期限・保管場所など）
  • ユーザー自身が操作した買い物リストのデータ

これらのデータはユーザー自身の Supabase プロジェクトにのみ保存されます。

3. 収集しない情報

  • 氏名・住所・電話番号などの個人識別情報
  • 音声データ・会話ログ（Alexa が処理する音声はアマゾン社の規約に従います）
  • 位置情報・デバイス情報

4. 第三者への提供

収集したデータを第三者へ販売・提供・共有することは一切ありません。 自然言語処理のために Google Gemini API を使用しますが、送信するのは在庫リストの情報のみであり、個人を特定する情報は含みません。

5. データの保管

データはユーザー自身が管理する Supabase プロジェクトに保管されます。 データの削除はユーザー自身が Supabase ダッシュボードから行うことができます。

6. お問い合わせ

本ポリシーに関するご質問は GitHub Issues からお問い合わせください。

1. Overview

"Housekeeper" is an Alexa skill for personal home inventory management. This skill is intended for personal use only and is not published on the Alexa Skills Store.

2. Information We Collect

The only information collected and stored by this skill is:

  • Inventory data entered by the user (product name, quantity, expiry date, storage location, etc.)
  • Shopping list data managed by the user

This data is stored solely in the user's own Supabase project.

3. Information We Do Not Collect

  • Personally identifiable information such as name, address, or phone number
  • Voice data or conversation logs (audio processed by Alexa is governed by Amazon's terms)
  • Location data or device information

4. Sharing with Third Parties

We do not sell, share, or provide collected data to any third parties. The Google Gemini API is used for natural language processing; only inventory list information is sent, and no personally identifiable information is included.

5. Data Storage

Data is stored in the user's own Supabase project under their control. Users can delete their data at any time through the Supabase dashboard.

6. Contact

For questions regarding this policy, please contact us via GitHub Issues.
