# Feature Spec: i18n (ja / en)

## 概要

UI の全文言を日本語 / 英語で切替可能にする。
デフォルトは `ja`、ブラウザ言語が `en*` ならフォールバックで `en`、設定があれば設定優先。

## ライブラリ

- `react-i18next`
- `i18next-browser-languagedetector`（ブラウザ言語を初期検出）
- 名前空間別 JSON: `src/locales/{ja,en}/{common,items,auth,settings,shopping,stats,notifications}.json`

## ユーザーストーリー

- 設定画面で言語を切替できる
- 切替は即時反映（リロード不要）
- 日付 / 数値も locale に従って表示される

## 実装方針

```ts
// src/lib/i18n.ts
import i18n from "i18next";
import LanguageDetector from "i18next-browser-languagedetector";
import { initReactI18next } from "react-i18next";

i18n
  .use(LanguageDetector)
  .use(initReactI18next)
  .init({
    fallbackLng: "ja",
    supportedLngs: ["ja", "en"],
    ns: ["common", "items", "auth", "settings", "shopping", "stats", "notifications"],
    defaultNS: "common",
    interpolation: { escapeValue: false },
    resources: { /* ja, en の import */ },
  });
```

設定変更は `useUserSettings` で `language` を更新 → effect で `i18n.changeLanguage(lang)` を呼ぶ。

## 文言抽出ルール

- すべての可視文字列は `t("namespace.key")` 形式
- 動的補間は `t("items.expiringIn", { count: n })` のような ICU 風キーで管理
- 日付は `Intl.DateTimeFormat(locale)`、数値は `Intl.NumberFormat(locale)`

## 単位の locale

- 単位（mL / g / 個 / 枚）は表示時に locale 別の表記をマッピングする
  - `units.mL.short`: `mL` / `mL`
  - `units.piece.short`: `個` / `pcs`
- 単位はテキストではなく **キー**で保持することを v1.x で検討（v1 は文字列のまま）

## エラー

- 翻訳キー欠損: `fallbackLng` で日本語にフォールバック、コンソール警告

## v1 範囲

- 全画面の ja/en リソース
- LanguageToggle atom
- 設定画面で切替
